# `friday setup`

Initialize `~/.friday/friday.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
friday setup
friday setup --workspace ~/friday
```

To run the wizard via setup:

```bash
friday setup --wizard
```
